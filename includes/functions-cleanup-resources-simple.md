## <a name="clean-up-resources"></a>Limpar recursos

Depois que você terminar de trabalhar com os recursos criados por este artigo, use o seguinte comando para excluí-los da sua assinatura:

```azurecli-interactive
az group delete --name myResourceGroup
```

Quando solicitado, digite `y`. Este comando remove o `myResourceGroup` grupo de recursos e todos os recursos que pertencem a ela.
