# <a name="azure-netapp-files-documentation"></a>Documentação do Azure NetApp Files

Saiba como usar o Azure NetApp Files para criar e gerenciar contas, pools de capacidade e volumes do NetApp; configurar a política de exportação; montar um volume para uma máquina virtual e gerenciar instantâneos.

> [!IMPORTANT] 
> Todas as tarefas descritas nesta documentação exigem que você faça parte do programa de Visualização Pública e esteja na lista de permissões para acessar o Provedor de recursos do Azure Microsoft.NetApp. Para obter mais detalhes sobre como ingressar no programa de Visualização Pública, confira a [Página de inscrição de versão prévia pública do Azure NetApp Files](https://aka.ms/nfspublicpreview).
