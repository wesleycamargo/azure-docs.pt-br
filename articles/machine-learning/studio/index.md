# <a name="azure-machine-learning-studio-documentation"></a>Documentação do Azure Machine Learning Studio

O Azure Machine Learning Studio é uma ferramenta colaborativa do tipo "arrastar e soltar", que você pode usar para criar, testar e implantar soluções de análise preditiva em seus dados.
Tutoriais, vídeos e modelos de exemplo mostram como usar o Studio para criar e implantar modelos de aprendizado de máquina.
