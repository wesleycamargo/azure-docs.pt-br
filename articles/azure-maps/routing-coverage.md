# <a name="azure-maps-routing-coverage"></a>Cobertura de rotas nos Mapas do Azure

Este artigo fornece informações de cobertura para rota nos Mapas do Azure. Quando você pesquisa a rota ideal do local A para B, mapas do Azure pode fornecer tempos de viagem altamente precisos, live instruções de informações e a rota de viagem atualizado. A rota levará em termos de pesquisa adicionais de conta, como tráfego atual, o tipo de veículo e condições para evitar. A capacidade de otimizar a rota depende da região, como mapas do Azure tem vários níveis de informações e a precisão para diferentes regiões. A tabela a seguir lista as regiões e que tipo de informação que você pode solicitar para eles.
 
Fazer check-out de cobertura para [ **Geocodificação**](geocoding-coverage.md).       
Fazer check-out de cobertura para [ **Tráfego**](traffic-coverage.md).  
Fazer check-out de cobertura para [ **Renderização**](render-coverage.md).
<br>  
**Legenda**

| Símbolo             | Significado                                |
|--------------------|----------------------------------------|

| Ø                  | País/região fornecido com dados simplificados. |
| País/região faltando | Dados de país/região não são fornecidos.          |
