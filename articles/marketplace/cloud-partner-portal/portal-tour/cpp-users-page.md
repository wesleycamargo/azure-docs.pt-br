# <a name="users-page"></a>Página Usuários

A página [**Usuários**](https://cloudpartner.azure.com/#users) exibe uma lista de todos os usuários registrados do Portal do Microsoft Cloud Partner.  As colunas incluem o endereço de e-mail de cada usuário, a função do portal, o tipo de oferta e a organização de publicação. Os resultados podem ser ordenados por coluna ou filtrados em uma cadeia de caracteres especificada.  

Esta página também exibe um botão **Adicionar usuário** para adicionar usuários à sua organização de publicação.  Para mais informações, consulte [Gerenciando usuários no Portal do Microsoft Cloud Partner](./../../cloud-partner-portal-orig/cloud-partner-portal-manage-users.md).
