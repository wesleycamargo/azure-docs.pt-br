# <a name="azure-data-factory-documentation"></a>Documentação do Azure Data Factory

O Azure Data Factory é um serviço de integração de dados baseado em nuvem que automatiza a movimentação e a transformação dos dados. Esse serviço cria soluções de integração de dados que podem receber dados de vários repositórios, transformar e processar os dados e publicar os dados resultantes de volta nos repositórios de dados.
