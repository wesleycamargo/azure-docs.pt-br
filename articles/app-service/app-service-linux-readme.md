# <a name="web-app-on-linux-documentation"></a>Documentação do Aplicativo Web no Linux
Atualmente, o Aplicativo Web no Linux está em Visualização Pública e permite que os clientes executem seus aplicativos Web nativamente em uma plataforma Linux. Isso permite uma melhor compatibilidade de aplicativos para determinados tipos de aplicativos e facilita a migração de aplicativos Web existentes hospedados em uma plataforma Linux em outro lugar nos Serviços de Aplicativo do Azure.

Os clientes podem criar aplicativos Web no Linux somente em um Plano do Serviço de Aplicativo habilitado para Linux, e esse Plano do Serviço de Aplicativo só pode hospedar aplicativos Web baseados em Linux. 

> O preço da Visualização Pública para o Aplicativo Web no Linux é 50% do preço comum e o preço é calculado reduzindo pela metade o número de horas de computação consumidas por seu Plano do Serviço de Aplicativo.
> 
> Nenhum SLA é oferecido durante o período de visualização pública.
